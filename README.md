# calculator.html 
📌 Project Overview

This is a fully functional and responsive Calculator Web Application developed using HTML, CSS, and JavaScript. The application is designed with a modern dark theme and smooth UI interactions to enhance user experience.

🚀 Features

Basic arithmetic operations (+, −, ×, ÷)

Percentage (%) calculation

Positive/Negative (+/−) toggle

Clear (AC) function

Decimal number support

Responsive and clean layout

Interactive hover and button animations

Gradient background with aesthetic design

🛠️ Technologies Used

HTML5 – Structure

CSS3 – Styling, gradients, layout design

JavaScript (ES6) – Logic implementation and event handling

🎯 Learning Outcomes

Improved understanding of DOM manipulation

Strengthened JavaScript logic building

Practiced UI/UX design principles

Implemented real-time calculation handling
